# Bubble Math Trivia 🫧

A gamified math trivia game where you pop floating bubbles by solving addition, subtraction, multiplication, and division problems[cite: 1]. Test your speed and accuracy as the equations float toward the top of the screen!

## 🌟 Features

* **Four Game Modes:** Choose between Addition, Subtraction, Multiplication, and Division[cite: 1].
* **Dynamic Difficulty:** Each mode features 5 levels that scale in speed, spawn rate, and equation complexity as your score increases[cite: 1].
* **Streak System:** Build your streak to earn x2, x3, and x4 score multipliers[cite: 1].
* **High Scores:** Your highest scores for each mode are automatically saved locally in your browser[cite: 1].
* **Rich Audio & Visuals:** Built with a custom Web Audio API synthesizer for sound effects and an HTML5 Canvas particle system for satisfying bubble-bursting animations[cite: 1].

## 🎮 How to Play

1. Select your preferred math operation from the main menu[cite: 1].
2. Click **Start Popping** to begin[cite: 1].
3. As bubbles float up the screen, solve the math equation written inside them[cite: 1].
4. Type your answer into the input box at the bottom of the screen. If you are correct, the bubble will automatically pop![cite: 1].
5. Do not let the bubbles escape the top of the screen! You only have 3 lives (hearts)[cite: 1].
6. Getting an answer wrong resets your streak multiplier[cite: 1].

## 🛠️ Technologies Used

* **HTML5:** Game structure and semantic layout[cite: 1].
* **CSS3:** Custom styling, glassmorphism UI, glowing animations, and responsive design[cite: 1].
* **Vanilla JavaScript:** Core game loops, math logic, particle rendering, and DOM manipulation (No external frameworks used)[cite: 1].
